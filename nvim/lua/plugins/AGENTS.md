## Build, Lint, and Test Commands

There are no build, lint, or test commands for this project. It is a Neovim configuration.

## Code Style Guidelines

*   **Language:** All code is written in Lua.
*   **Formatting:**
    *   Indentation is 2 spaces.
    *   Use double quotes for strings.
*   **Imports:** Use `require()` to import modules.
*   **Structure:** Files return a table that defines the plugin configuration.
*   **Naming Conventions:**
    *   Variables and functions are `snake_case`.
    *   Files are named after the plugin they configure.
*   **Error Handling:** There is no explicit error handling.
*   **Types:** The project does not use a static type checker.
